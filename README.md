# Производственный навигатор

> Интеллектуальная система анализа рынка, поиска заказов и поддержки принятия решений для производственных предприятий.

**Девиз проекта**

> Находим не тендеры, а возможности для производства.

---

## Что умеет система

- 📋 Анализ производственного предприятия
- 🏭 Учет оборудования и производственных возможностей
- 🔍 Поиск государственных и коммерческих заказов
- 🤖 AI-рекомендации по развитию предприятия
- 📊 Аналитика рынка
- 📄 Работа с документацией
- 👥 CRM для клиентов и заказов

---

## Для кого

- небольшие производственные предприятия;
- мастерские с ЧПУ;
- металлообработка;
- машиностроение;
- изготовители нестандартного оборудования.
