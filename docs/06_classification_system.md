# Справочники
## Назначение

Данный документ определяет корпоративную систему классификации
Production Navigator Classification (PNC), используемую во всех
компонентах проекта: базе данных, API, бизнес-логике и пользовательском
интерфейсе.

## Принципы построения справочников

Справочники проекта строятся на основе международных промышленных
классификаторов (в первую очередь eCl@ss) с использованием ссылок
на соответствующие разделы ISO ICS.

Production Navigator Classification (PNC) не заменяет международные
классификаторы, а объединяет их в единую модель данных проекта, 
используется для обеспечения совместимости с международной промышленной
терминологией.

## Структура справочников

Во всех справочниках используются единые поля:

| Поле        | Назначение                                |
|-------------|-------------------------------------------|
| code        | Внутренний неизменяемый идентификатор PNC |
| name_ru     | Наименование на русском языке             |
| ics_section | Ссылка на Раздел ISO ICS (если применимо) |
| ref_system  | Внешний классификатор                     |
| ref_code    | Код во внешнем классификаторе             |  
| description | Описание области применения               |

Поле `code` является первичным бизнес-идентификатором и не изменяется
после публикации системы.

### Примечания

- `code` — основной идентификатор Production Navigator 
Classification (PNC), используемый в базе данных, API и 
программном коде.
- `ics_section` — ссылка на раздел International Classification
for Standards (ISO ICS), предназначена для сопоставления со 
стандартами и не является идентификатором оборудования.
- `ref_system` — внешний классификатор, с которым сопоставлено значение PNC.
- `ref_code` — код значения во внешнем классификаторе; для справочников на основе eCl@ss поле `ref_system` принимает значение `eCl@ss`.
