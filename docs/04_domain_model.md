# Модель данных

```text
EnterpriseProfile
├── Product

├── Technology

```

## EnterpriseProfile

Назначение:
Цифровой производственный профиль предприятия.

| Поле            | Тип     | Обязательное | Описание                 |
|-----------------|---------|--------------|--------------------------|
| id              | Integer | Да           | Идентификатор            |
| company_name    | String  | Да           | Наименование предприятия |
| production_type | Enum    | Да           | Серийное / Под заказ     |
| description     | Text    | Нет          | Описание предприятия     |

Связи:

EnterpriseProfile
├── Products
├── Equipment
├── Technologies
├── Materials
├── Infrastructure
├── Certificates
└── Resources

## Technology

Назначение: справочник производственных технологий и операций.

| Поле          | Тип        | Обязательное | Описание                                 |
|---------------|------------|--------------|------------------------------------------|
| id            | Integer    | Да           | Идентификатор технологии                 |
| name          | String     | Да           | Наименование технологии                  |
| category      | String/Enum| Да           | Категория технологии                     |
| description   | Text       | Нет          | Дополнительное описание                  |
| is_active     | Boolean    | Да           | Используется ли технология в справочнике |

## Примеры category:

```text

```

| Поле            | Тип     | Обязательное | Описание                                      |

| notes           | Text    | Нет          | Ограничения и особенности                     |

# Materials

## MaterialGroup

Назначение: иерархический справочник категорий и групп материалов.

| Поле       | Тип     | Обязательное | Описание                             |
|------------|---------|--------------|--------------------------------------|
| id         | Integer | Да           | Идентификатор группы                 |
| parent_id  | FK      | Нет          | Родительская группа                  |
| name       | String  | Да           | Наименование группы                  |
| description| Text    | Нет          | Дополнительное описание группы       |
| is_active  | Boolean | Да           | Используется ли группа в справочнике |

```text
Металлы
├── Черные металлы
│   ├── Конструкционные стали
│   ├── Инструментальные стали
│   └── Нержавеющие стали
└── Цветные металлы
    ├── Алюминиевые сплавы
    ├── Медные сплавы
    └── Титановые сплавы
```
#№ Material

Назначение: справочник конкретных материалов и их марок.

| Поле          | Тип     | Обязательное | Описание                               |
|---------------|---------|--------------|----------------------------------------|
| id            | Integer | Да           | Идентификатор материала                |
| group_id      | FK      | Да           | Группа материала                       |
| grade_name    | String  | Да           | Марка материала                        |
| standard_name | String  | Нет          | Стандарт или нормативный документ      |
| density       | Float   | Нет          | Плотность материала, кг/м³             |
| hardness_value| Float   | Нет          | Значение твердости                     |
| hardness_scale| String  | Нет          | Шкала твердости: HB, HRC, HV и другие  |
| description   | Text    | Нет          | Дополнительные свойства и примечания   |
| is_active     | Boolean | Да           | Используется ли материал в справочнике |

```text
Группа: Нержавеющие стали
Марка: 12Х18Н10Т
Стандарт: ГОСТ 5632
Плотность: 7900 кг/м³
Твердость: 179 HB
```
